# Auto-Moto-Touring

Auto/Moto Touring è il progetto per Laboratorio di Architetture Software e Sicurezza Informatica di Lorenzo Magni ed Eleonora Beatrice Rossi,
studenti della Sapienza.

Il portale permette a vari utenti di condividere itinerari e tragitti da percorrere in auto in moto al fine di condividere insieme, 
la passione per la guida.

Per poter sviluppare l'applicazione occorre:

1: installare Ruby sul proprio sistema, per windows l'installer https://github.com/oneclick/rubyinstaller2/releases/download/RubyInstaller-3.0.1-1/rubyinstaller-devkit-3.0.1-1-x64.exe 
poi esaeguirlo e completare tutta l'installazione, lasciare selezionate le spunte che vengono richieste
2: al momento di apertura della shell, sarà necessario installare tutte e 3 le dipendenze (1,2,3) e poi fare ENTER
3: sarà poi necessario isntallare YARN al seguente link, basterà scaricarlo ed installarlo:   https://classic.yarnpkg.com/latest.msi

4: poi bisognerà installare rails: aprire il terminale e digitare il seguente comando senza parentesi (gem install rails)

5: PRIMA DI PROCEDERE controllare le versioni di tutto:   ruby -v        rails --version   yarn --version

poi viene creato il progetto ruby.

Predisporre il proprio IDE (in questo caso VISUAL STUDIO CODE) con le estensioni utili al caso:
Bootstrap4, Font Awesome4, Font Awesome 5 Free & Pro snippets, Ashok Koyi
